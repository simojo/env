### Overview

**Bar**: [Polybar](https://github.com/polybar/polybar)  
**WW**: [bspwm](https://github.com/baskerville/bspwm)

**HotKeys**: [sxhkd](https://github.com/baskerville/sxhkd)  
**Shell**: [bash](https://www.gnu.org/software/bash/)
